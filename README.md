# Caesar Cipher (Python)

A simple **Caesar Cipher** implementation written in Python as part of the **100 Days of Code - The Complete Python Pro Bootcamp** course.

---

## 🇬🇧 English Version

### 📌 About

This project implements the classic **Caesar Cipher** encryption algorithm.

The program allows users to:

* Encrypt text
* Decrypt text
* Choose a custom shift value
* Encrypt and decrypt complete sentences
* Run the program multiple times without restarting

---

### ✨ Features

* Encode messages
* Decode messages
* Custom shift amount
* Input validation
* Modular project structure
* Clean and readable code

---

### 📂 Project Structure

.

### ▶️ Example

```text
Type 'encode' to encrypt, type 'decode' to decrypt:
encode

Type your message:
hello world

Type the shift number:
5

Here's the encoded result:
mjqqt btwqi
```
